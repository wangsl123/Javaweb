## JSp页面元素

1. 脚本Scriptlet
    - <%
           局部变量，Java代码
       %>
    - <%!
            全局变量，定义方法
    
       %>
    - <%= 输出表达式%>
```jsp
<%@ page language="java" contentType="text/html; charset=UTF-8"
         pageEncoding="UTF-8"%>
<!DOCTYPE html PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
<html>
<body>
    <h2>Hello World!</h2>

    <%!
        public String bookName;
        public void init(){
            bookName = "java书";
        }
    %>
    <%
        String name = "hahaha";
        out.println(name);
        init();
    %>

    <%="hello"+bookName%>
</body>
</html>
```

2. 指令
- page指令：
```java
<%@ page language="java" contentType="text/html; charset=UTF-8"
         pageEncoding="UTF-8"%>
```
