## round

**四舍五入**

```SQL
select round(-1.67);
select round(-1.678,2);
```

## ceil

**向上取整**

```SQL
select ceil(-1.2);
```

## floor

**向下取整**

```SQL
select floor(-9.9);
```

## truncate

**截断**

```SQL
select truncate(1.89999,2);
```

## mod

**mod(a,b)     ==    a-a/b*b**

```sql
select mod(10,-3);
```
